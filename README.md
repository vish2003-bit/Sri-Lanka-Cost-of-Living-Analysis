## 📊 Project Overview

This project explores the cost of living in Sri Lanka using data analysis and visualization techniques.

The objective is to identify spending patterns, analyze economic trends, and transform raw data into meaningful insights that can support informed decision-making.

---

## 🎯 Objectives

- Analyze living expenses in Sri Lanka
- Identify key spending categories
- Explore economic trends and patterns
- Create meaningful visualizations
- Develop practical data analytics skills

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
